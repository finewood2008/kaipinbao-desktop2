   - 大号标题「开品宝」
   - 渐变色副标题「您的第二开发部」
   - 功能点用点号分隔，单行展示
   - 橙色/渐变 CTA 按钮

3. **功能卡片**：玻璃拟态风格，hover 时发光效果
4. **客户类型卡片**：突出电商公司和外贸工厂两类核心用户

## 技术实现要点

1. **响应式设计**：移动端适配，功能点自动换行
2. **动画效果**：使用 framer-motion 实现入场动画
3. **渐变效果**：复用现有 CSS 变量（text-gradient、bg-gradient-primary）
4. **毛玻璃效果**：使用现有 glass 类
